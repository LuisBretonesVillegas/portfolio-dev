Opening paragraph. One or two sentences setting the context — what problem you ran into, what you were trying to do, or why this is worth writing down. No heading needed here.

## Section heading

Body of the section. Keep paragraphs short.

For inline code use backticks: `command --flag`.

For a code block:

```bash
# comment
command --flag value
```

For a note or tip:

> This is a blockquote. Useful for warnings or side notes.
